# React useRef - 10 Ejercicios

Este repositorio contiene una colección de 10 ejercicios prácticos diseñados para aprender y dominar el hook `useRef` en React.

## 🚀 Objetivo

Comprender el uso real de `useRef` en escenarios comunes de desarrollo frontend, incluyendo:

- Referencias al DOM
- Manejo de inputs no controlados
- Persistencia de valores sin re-render
- Control de timers (setInterval / clearInterval)
- Detección de renders
- Solución de closure bugs
- Scroll automático tipo chat
- Medición de elementos del DOM
- Comparación entre useRef y useState

## 🧪 Tecnologías

- React (CDN)
- JavaScript (ES6)
- HTML

## 📚 Estructura

Cada ejercicio está diseñado para resolver un problema específico y aumentar progresivamente la dificultad.

## 🎯 Aprendizajes clave

- Cuándo usar `useRef` vs `useState`
- Cómo evitar renders innecesarios
- Cómo interactuar directamente con el DOM
- Cómo manejar efectos secundarios correctamente
